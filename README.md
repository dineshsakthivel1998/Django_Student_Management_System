# Django Student Management System

A web-based Student Management System built using the Django framework.  
The application allows authenticated users to manage student records with full CRUD functionality.

## Features

- User Registration
- User Login & Logout Authentication
- Add Student Records
- View Student List
- View Student Details
- Update Student Information
- Delete Student Records
- Responsive UI using Bootstrap
- Deployed on Render

## Technologies Used

- Python
- Django
- HTML
- Bootstrap
- SQLite
- Git & GitHub
